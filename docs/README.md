# human · Documentation

**human** is a Zero-Knowledge proof-of-personhood system on Stellar: verify that someone is a real, unique human **without exposing personal data on-chain**, then build privacy-preserving applications on top.

## Sync with GitBook

This folder is designed for [GitBook Git Sync](https://docs.gitbook.com/integrations/git-sync):

1. Create a GitBook account and a new Space named `human`.
2. Connect this repository (`obsidian-vault-zk`).
3. Set the **root directory** to `docs/`.
4. GitBook reads navigation from [`SUMMARY.md`](SUMMARY.md).
